Este plugin permite añadir videos de youtube y mostrar una imagen de precarga del video.

== Descripción ==
Plugin para colocar en las paginas de WordPress videos de youtube y no cargarlos hasta que el usuario haga click.
En la primera carga lo que se vera es una imagen del vídeo de youtube.
